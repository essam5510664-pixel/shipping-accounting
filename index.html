كيف تربط GitHub مع Netlify لنشر موقعك مجاناً

سأشرح لك خطوة بخطوة كيفية رفع مشروعك إلى GitHub ثم ربطه مع Netlify للحصول على موقع ويب مجاني.

1

أنشئ حساب GitHub

اذهب إلى github.com وأنشئ حساباً مجانياً.

إنشاء حساب GitHub
2

أنشئ مستودعاً جديداً

بعد تسجيل الدخول، انقر على "New repository" لإنشاء مستودع جديد لمشروعك.

// اسم المستودع: shipping-accounting
// نوعه: Public
// اختر: Add a README file
3

ارفع ملفاتك

انقر على "Add file" ثم "Upload files" لرفع ملف index.html وملفاتك الأخرى.

// يمكنك رفع الملفات عن طريق السحب والإفلات
// أو استخدام الأمر git push إذا كنت تعرف Git
4

انسخ عنوان المستودع

انسخ عنوان URL الخاص بمستودعك من شريط العنوان في المتصفح.

https://github.com/اسم-المستخدم/اسم-المستودع
5

سجل الدخول إلى Netlify

اذهب إلى netlify.com وسجل الدخول بحسابك.

إنشاء حساب Netlify
6

استورد من GitHub

انقر على "Import from Git" ثم اختر GitHub وأعط الصلاحيات اللازمة.

اختر المستودع الذي أنشأته من القائمة.

7

اضبط إعدادات النشر

اتبع التعليمات واضغط على "Deploy site" وسيتم نشر موقعك تلقائياً.

ستحصل على رابط مجاني مثل: your-site-name.netlify.app

8

التحديثات التلقائية

الآن أي تحديث ت pushه إلى GitHub سينتقل تلقائياً إلى موقعك على Netlify.

نصائح مهمة:

  • تأكد من أن المستودع عام (Public) للحساب المجاني
  • اسم الملف الرئيسي يجب أن يكون index.html
  • يمكنك إضافة نطاق مخصص لاحقاً إذا أردت
ابدأ الآن بأنشئ مستودع جديد اذهب إلى Netlify
